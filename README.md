# Line Follower Robot

A Line Follower Robot designed to autonomously detect and follow a predefined path using infrared (IR) sensors and a motor driver control system. The robot continuously senses the line position and adjusts its movement in real time to maintain accurate tracking. This project demonstrates the practical implementation of embedded systems, sensor interfacing, motor control, and autonomous navigation concepts commonly used in robotics and industrial automation.

## Features

* Real-time line detection using IR sensors
* Autonomous path-following capability
* Differential motor speed control for steering
* Compact and efficient embedded design
* Suitable for robotics learning and competitions

## Hardware Components

* Microcontroller
* IR Sensor Module
* Motor Driver (L293D/L298N)
* DC Geared Motors
* Robot Chassis
* Battery Pack

## Applications

* Autonomous mobile robots
* Industrial material handling systems
* Automated guided vehicles (AGVs)
* Robotics education and research

## Project Objective

The primary objective of this project is to develop an intelligent robotic system capable of following a designated path with minimal human intervention while demonstrating the fundamentals of sensing, decision-making, and motor control in embedded systems.
